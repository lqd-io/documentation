## Liquid Documentation

We use Jekyll to write and update our documentation articles.

Quick-start instructions:

```ruby
~ $ gem install jekyll
~ $ cd project-folder
~/project-folder $ jekyll serve
# => Now browse to http://localhost:4000
```
